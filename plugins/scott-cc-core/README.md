# scott-cc-core

> Essential development tools for everyday coding

## What's Included

### Agents (3)
- **api-debugger** - Debug APIs, Python backends, and JavaScript frontends with integrated browser testing
- **system-architect** - Design scalable system architecture with focus on maintainability
- **refactoring-expert** - Improve code quality and reduce technical debt

### Skills (2)
- **context7** - Retrieve up-to-date documentation for any programming library
- **python-simplifier** - Simplify and refine Python code for clarity and consistency

## When to Use

**Install this if you**: Do general development work and need essential debugging, architecture, and refactoring tools.

**Don't install if you**: Want specialized tools only (security, testing, etc.) - install those specific plugins instead.

## Quick Start

```bash
# API debugging
Ask: "Debug this API endpoint returning 500 errors"

# Architecture design
Ask: "Design a scalable architecture for a multi-tenant SaaS app"

# Code refactoring
Ask: "Refactor this class to follow SOLID principles"

# Documentation lookup
/context7 How to use React useEffect cleanup functions
```
